# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inspiring community for all. We pledge that everyone participating in the PISSM project and community will experience a harassment-free environment.

## Our Standards

Examples of behavior that contribute to creating a positive environment include:

- **Being respectful**: Treat all members with courtesy and respect
- **Being constructive**: Provide helpful feedback and suggestions
- **Being inclusive**: Welcome newcomers and encourage participation
- **Being patient**: Remember that people have different backgrounds and expertise
- **Being professional**: Keep discussions focused and on-topic

Examples of unacceptable behavior include:

- Harassment, insults, or derogatory comments
- Discrimination based on protected characteristics
- Unwelcome sexual advances or attention
- Intimidation or threats
- Advocating for or encouraging any unacceptable behavior

## Scope

This Code of Conduct applies to:
- All project spaces (code repositories, issue trackers, discussions)
- Project-related communication channels
- Events and meetings related to the project
- Both public and private interactions

## Enforcement

### Reporting Issues

If you experience or witness unacceptable behavior, please report it by:

1. **Direct contact**: Reach out to project maintainers privately
2. **GitHub**: Use private messages to project maintainers
3. **Email**: Contact maintainers at the project email (if provided)

Please include:
- Description of the incident
- Context and background
- Who was involved (if comfortable sharing)
- How the incident affected you

### Response

- All reports will be reviewed fairly and objectively
- Investigation will be conducted promptly
- The privacy of reporters will be protected
- Appropriate action will be taken based on findings
- We will follow up with both parties when possible

### Consequences

Project maintainers who do not follow the Code of Conduct may face consequences including:
- Private warnings
- Temporary restrictions on participation
- Permanent removal from the project

## Guidelines for Maintainers

- Set an example by following the Code of Conduct
- Be responsive to reports of violations
- Address violations promptly and fairly
- Maintain confidentiality when possible
- Support and mentor community members

## Acknowledgments

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.0.

## Questions?

For questions about this Code of Conduct or how to report violations, please reach out to the project maintainers.
